# Автоматизация синхронизации KeePassXC с Google Drive через rclone

Данный проект предоставляет автоматизированное решение для настройки синхронизации базы паролей KeePassXC с Google Drive через rclone.

## Обзор

Этот проект включает в себя:
- Скрипты автоматизации для установки и настройки rclone
- Скрипты для настройки автозапуска через systemd
- Утилиты для управления монтированием Google Drive
- Документация по всем аспектам настройки

## Структура проекта

```
rclone-keepassxc-sync/
├── docs/                    # Документация
│   ├── setup-guide.md       # Подробное руководство по настройке
│   └── usage-guide.md       # Руководство по использованию скриптов
├── scripts/                 # Скрипты автоматизации
│   ├── install-rclone.sh    # Установка rclone
│   ├── configure-rclone.sh  # Настройка rclone для Google Drive
│   ├── setup-systemd.sh     # Настройка автозапуска через systemd
│   └── mount-manager.sh     # Управление монтированием/размонтированием
├── systemd/                 # Файлы сервисов systemd
│   └── rclone-gdrive.service
├── install.sh              # Главный скрипт установки
├── README.md               # Общее описание проекта
```

## Требования

- Arch Linux (или совместимая система с pacman)
- KeePassXC
- Google аккаунт

## Безопасность

**ВАЖНО**: Всегда закрывайте базу KeePassXC перед выключением компьютера.
Рекомендуется использовать функцию автосохранения в KeePassXC.
Делайте резервные копии базы раз в месяц.

## Использование на других устройствах

- Установите KeePassXC/KeePassDX
- Настройте доступ к тому же файлу в Google Drive
- Используйте **тот же мастер-пароль**
